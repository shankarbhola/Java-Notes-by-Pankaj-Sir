Functional Interface
--------------------
In an Functional Interface their should be exactly 1 incomplete method in it.

@FunctionalInterface
public interface A { //Error
	
}
---
@FunctionalInterface
public interface A { //Error
	public void test1();
	public void test2();
	
}
---
@FunctionalInterface
public interface A { 
	public void test1();
	
}

public class B implements A{

	@Override
	public void test1() {
		System.out.println("From Test1");
	}
	public static void main(String[] args) {
		B b1 = new B();
		b1.test1();
	}
}

default Keyword
---------------
This was introduced in java versio 8. Using default keyword we can create complete methods in an interface.

@FunctionalInterface
public interface A { 
	public void test1();
	default public void test2() {
		System.out.println("From Test2");
	}
}

public class B implements A{

	@Override
	public void test1() {
		System.out.println("From Test1");
	}
	public static void main(String[] args) {
		B b1 = new B();
		b1.test1();
		b1.test2();
	}
}

---

@FunctionalInterface
public interface A {

		System.out.println("From Test2");
	}
	default public void test3() {

@FunctionalInterface
public interface A { 
	public void test1();
	default public int test2() {
		return 100;
	}
}

public class B implements A{

	@Override
	public void test1() {
		System.out.println("From Test1");
	}
	public static void main(String[] args) {
		B b1 = new B();
		b1.test1();
		int val = b1.test2();
		System.out.println(val);
	}
}

Lambda Expression
-----------------
This was introduced in java version 8. It helps to reduce the number pof lines of code in program.

@FunctionalInterface
public interface A { 
	public void test1();
	
}

public class B{

	public static void main(String[] args) {
		A a1=()->{
			System.out.println(100);
		};
		a1.test1();
	}
}

-----
@FunctionalInterface
public interface A { 
	public void test1(int x);
	
}

public class B{

	public static void main(String[] args) {
		A a1=(int x)->{
			System.out.println(x);
		};
		a1.test1(100);
	}
}
-----
@FunctionalInterface
public interface A { 
	public void test1(int x, int y);
	
}

public class B{

	public static void main(String[] args) {
		A a1=(int a, int b)->{
			System.out.println(a);
			System.out.println(b);
		};
		a1.test1(100,200);
	}
}
-----
@FunctionalInterface
public interface A { 
	public void test1(int x, int y);
	default public void test2() {
		System.out.println(30);
	}
}

public class B{

	public static void main(String[] args) {
		A a1=(int a, int b)->{
			System.out.println(a);
			System.out.println(b);
		};
		a1.test1(10,20);
		a1.test2();
	}
}
